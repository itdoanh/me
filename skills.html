Kỹ năng Công nghệ

Bộ công cụ và năng lực kỹ thuật tôi sử dụng để kiến tạo những hệ thống chất lượng cao.

AI-Assisted Development

Sử dụng cấp độ chuyên gia các công cụ Google Antigravity, Cursor, GitHub Copilot. Áp dụng luồng Vibe Coding / Context-Driven qua việc tinh chỉnh cấu hình ngữ cảnh (.cursorrules), kết hợp đa mô hình (Gemini, Claude, GPT) để tự động hóa viết logic phức tạp.

Google Antigravity Cursor GitHub Copilot Vibe Coding Gemini Claude GPT

Backend & Database

Xây dựng hệ thống và API hiệu năng cao với Go (cốt lõi), Laravel, Node.js. Thiết kế schema và tối ưu truy vấn cho CSDL quan hệ (MySQL, SQL Server) và CSDL phân tán chịu tải cao (ScyllaDB).

Go Laravel Node.js RESTful API MySQL SQL Server ScyllaDB

System & Core

Lập trình hệ thống (C/C++, Windows API). Nắm vững tư duy quản trị Linux Server, cấu hình mạng lưới (Networking) và triển khai bảo mật hạ tầng thực tế.

C/C++ Windows API Linux Server Networking Security Port Forwarding

Frontend & UI/UX

Thành thạo HTML5, CSS3, JavaScript (ES6+). Tư duy thiết kế lấy người dùng làm trung tâm (User-Centric) qua Figma, Photoshop, Illustrator. Kỹ năng hậu kỳ Media (Premiere, After Effects).
